<samp><b>tinyRipple's eslint config preset</b></samp>

## USAGE

### Install

```bash
# npm
npm install -D eslint @denverzh/eslint-config

# yarn
yarn add -D eslint @denverzh/eslint-config

#pnpm
pnpm add -D eslint @denverzh/eslint-config
```

### Config `.eslintrc`

```json
{
  "extends": "@denverzh"
}
```
