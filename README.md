## USAGE

### Install

```bash
# npm
npm install -D eslint @isdhzh/eslint-config

# yarn
yarn add -D eslint @isdhzh/eslint-config

#pnpm
pnpm add -D eslint @isdhzh/eslint-config
```

### Config `.eslintrc`

```json
{
  "extends": "@isdhzh"
}
```
